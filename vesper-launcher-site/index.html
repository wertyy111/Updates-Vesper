{ faq }

Ответы перед запуском

Без декоративных блоков: только ответы на главные вопросы перед запуском.

Какие версии Minecraft планируются в первую очередь?

Сначала логично вывести vanilla, а затем дать отдельные профили под Forge и Fabric, чтобы пользователь не путал сборки.

Будут ли отдельные профили под модпаки?

Да. В текущей структуре интерфейса уже есть место для хранения разных пресетов, зависимостей и статусов загрузки.

Что с безопасностью и понятностью для новичков?

Русский интерфейс, честные статусы, внятные предупреждения и отсутствие ложных CTA делают вход заметно спокойнее.

Можно ли будет читать changelog прямо в лаунчере?

Да, под это уже заложен визуальный паттерн карточек: новости, патчи, срочные анонсы и релизные заметки.

Зачем нужен отдельный экран сообщества?

Чтобы пользователь видел, где получать апдейты, обратную связь и тестовые сборки, а не искал их по случайным сообщениям.
