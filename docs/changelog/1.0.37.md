# Vesper Launcher 1.0.37

- Добавлен первый рабочий VesperNet Overlay поверх текущего relay для подключения к миру друга.
- Лаунчер умеет переключаться на overlay-режим для host и guest, с fallback на старый Vesper Relay.
- Добавлены подробные диагностические логи для VesperNet в лаунчер и сервис, чтобы быстро разбирать проблемы подключения.
- Улучшено открытие меню модов: панель показывается сразу, а каталог грузится с индикатором загрузки.
