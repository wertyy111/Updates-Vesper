# Vesper Launcher 1.0.38

- Кнопка подключения к другу теперь появляется только когда друг реально открыл мир для сети.
- Улучшена стабильность подключения к миру друга: лаунчер удерживает активную LAN-сессию дольше и быстрее подхватывает входящие подключения host-стороны.
- Меню модов открывается быстрее и показывает индикатор загрузки, пока каталог ещё подгружается.
- Улучшен 3D-предпросмотр скина внутри лаунчера.
